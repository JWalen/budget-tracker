# Contributing

Thank you for your interest in Budget Tracker.

This project is **source-available for viewing only**. The code is published for
transparency and personal, non-commercial use — it is **not** open source (see
the [LICENSE](LICENSE) file).

At this time:

- **Pull requests are not accepted.**
- **Forks, modifications, and redistribution are not permitted** under the
  license.
- The project is maintained solely by the author.

If you've found a bug or have a suggestion, you're welcome to open a
[GitHub Issue](https://github.com/JWalen/budget-tracker/issues) — feedback is
appreciated, even though outside code contributions aren't accepted.

For any use beyond personal, non-commercial use — including commercial use,
modification, or redistribution — please contact the author to request
permission.
